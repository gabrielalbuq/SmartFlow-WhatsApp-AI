# SmartFlow-WhatsApp-AI
Plataforma SaaS escalável para gestão de chatbots inteligentes via WhatsApp. Desenvolvido em Python com suporte a IA multimodal (texto e áudio) e integração com banco de dados relacional. Estrutura flexível e modular, projetada para processar interações naturais e gerenciar múltiplas instâncias de atendimento de forma automatizada.
